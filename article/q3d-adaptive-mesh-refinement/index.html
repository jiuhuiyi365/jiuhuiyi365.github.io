1. 为什么要自适应网格加密

在有限元电磁仿真中，网格密度直接决定求解精度。但全域统一加密会带来巨大的计算开销——仿真时间随网格数呈超线性增长。理想的做法是：只在电流密度变化剧烈的地方加密网格，平滑区域保持粗网格。

这就是自适应网格加密（Adaptive Mesh Refinement）的核心思想：

  1. 在粗网格上求解一次
  2. 评估每个三角面单元的误差
  3. 在误差大的区域细分网格
  4. 在新网格上重新求解
  5. 重复直到收敛

Q3D Extractor 使用 Raviart-Thomas（RT）边缘元离散电流密度场。在这种离散下，电流密度的不连续性可以直接作为误差指标——如果电流穿过单元边界时发生了跳变，说明解不够好。

2. 自适应加密的整体流程

每一轮自适应迭代（Adaptive Pass）中，求解器按以下步骤执行：

  1. 求解器设置：加载网格、分配求解器数据结构
  2. 矩阵组装：构建 (R + jωL) 矩阵
  3. PARDISO 求解：求解 (R + jωL)·I = V，得到每条边的电流系数
  4. 误差评估（本文重点）：对所有三角面计算误差
  5. 排序选择：按误差排序，选出需要细化的单元
  6. 网格细化：对选中的三角面进行细分
  7. 收敛判断：如果 DeltaNorm 小于阈值，停止迭代
典型参数
在一次 Q3D 仿真中，初始网格约 4650 个三角面，经过 5 轮自适应迭代后增长到约 6814 个三角面，每次细化约 10% 的单元。

3. 电流密度 J 的计算

3.1 PARDISO 求解与边电流系数

求解器组装稀疏矩阵方程：

(R + jωL) · I = V

其中 R 是电阻矩阵（对角），L 是电感矩阵（非对角，由 BEM 积分计算），V 是激励向量（端口边为 1V，其他为 0V）。

PARDISO 求解后得到 边电流系数 I，每个三角面有 3 个系数，分别对应 3 条边：

I(e) = [I₀, I₁, I₂]

每个系数是复数，实部对应传导电流，虚部对应位移电流。

3.2 从系数到 J：Raviart-Thomas 基函数重构

电流密度 J 在三角面单元内由 3 个 Raviart-Thomas 基函数重构：

J(e) = I₀ · w₀ + I₁ · w₁ + I₂ · w₂

对于边 i（设其两个端点为 vₐ, v_b，对顶点为 vopp），RT 基函数在单元重心处取值为：

wi(center) = (center − vopp) / (2·area)

展开后：

J(e) = Σi=0,1,2 Ii · (center − vopp,i) / (2·area)

其中 center 为三角形重心坐标，area 为三角形面积，vopp,i 为边 i 的对顶点。完整计算为：

// 对三角面的每条边 (i = 0, 1, 2):
// 边 0: 对顶点 = n₂, 边向量 = (n₀→n₁)
// 边 1: 对顶点 = n₀, 边向量 = (n₁→n₂)
// 边 2: 对顶点 = n₁, 边向量 = (n₂→n₀)

dx = center_x - vopp_x;
dy = center_y - vopp_y;
dz = center_z - vopp_z;

Jx += real(I_i) × dx / (2 × area);
Jy += real(I_i) × dy / (2 × area);
Jz += real(I_i) × dz / (2 × area);
注意
在 DC 模式下，虚部为 0，只取实部。在交流模式下，电流可能分布在实部或虚部，取非零的那部分即可。

4. 误差公式：error = area × (v43² + v44²)

4.1 每条边的连续性检查

对每个三角面，遍历其 3 条边。对于每条边：

  1. 找到共享这条边的邻居三角面
  2. 计算自三角面的 J 在边方向上的投影
  3. 计算邻居三角面的 J 在边方向上的投影（乘以法向量夹角绝对值 |dot|）
  4. 将两者 相加（代码中是加法，不是减法），乘以边长，累加到 v43/v44

法向量夹角修正 |dot| 的意义：两个三角面可能不在同一平面上（如曲面或棱边处），直接用 J_neigh 投影会忽略几何弯曲带来的方向差异。乘以 |norm_self · norm_neigh| 后，弯曲程度越大修正越大，确保弯曲区域的误差不被低估。

// 对每条边 j (0, 1, 2):
vA = 顶点[j], vB = 顶点[j+1 mod 3]

// 边方向单位向量
edge_vec = vB - vA
edge_len = |edge_vec|
dir = edge_vec / edge_len = (nx, ny, nz)

// 自三角面的 J 投影到边方向
J_self_proj = J_self · dir

// 邻居三角面的 J 投影到边方向（乘以法向量夹角绝对值）
dot_norm = |norm_self · norm_neigh|
J_neigh_proj = (J_neigh · dir) × dot_norm

// ★ 累加：代码中是加法不是减法
//    因为 J_self 和 J_neigh 指向边时方向相反，
//    如果解精确，J_self_proj + J_neigh_proj ≈ 0
v43 += (J_self_proj + J_neigh_proj) × edge_len
v44 += 0  // DC 模式下虚部为 0

4.2 J_jump：跨边跳变量

如果电流密度在共享边上完全连续，那么：

Jself · dir ≈ −Jneigh · dir

两者互相抵消，v43 ≈ 0，误差 ≈ 0。

如果电流密度不连续（解不够精确），两者相加不为零，v43 很大，误差很大——这个区域需要细化。

物理含义
误差本质上是 电流密度法向分量在单元边界上的跳变。在 RT 边缘元中，J 的法向分量应该是跨边连续的。跳变越大，误差越大。

4.3 最终公式

error(e) = geometric_area(e) × (v43² + v44²)

其中：

变量含义计算方式
geometric_area(e)三角面面积½ × |(v₁−v₀) × (v₂−v₀)|
v433 条边的 J 跳变实部累加Σ (J_self_proj + |dot|·J_neigh_proj) × edge_len
v443 条边的 J 跳变虚部累加（DC = 0）同上，取虚部

5. 哪些元素会被加密

5.1 对数分桶

计算出所有三角面的 error(e) 后，求解器按对数尺度将它们分配到 20 个桶中：

bucket = ⌊log₁₀(error(e) / max_error) × (−2)⌋

桶 0 包含误差最大的单元（error ≈ max_error），桶 19 包含误差最小的单元（error ≈ max_error × 10−9.5）。

桶编号误差范围说明
0max_error ~ max_error/√10误差最大，优先细化
1max_error/√10 ~ max_error/10需要细化
⋯⋯⋯
19误差极小通常不会被选中

同桶内的所有单元在 ADP 文件中共享同一个桶化误差值（即桶的代表值），而非原始误差。

5.2 前 PercentRefine%

求解器计算目标细化数：

// DC 模式
target = ceil(totalTriangles × PercentRefine × 1.25 / 100)

// 非 DC 模式
target = ceil(totalTriangles × PercentRefine × 0.5 / 100)

然后从桶 0 开始，逐桶取出单元，直到凑够 target 个。这些单元会被标记为"需要加密"，写入 ADP 文件供网格细化使用。

典型情况下 PercentRefine ≈ 10%，即每轮细化约 10% 的三角面。

6. 收敛条件

自适应迭代的收敛由 DeltaNorm 判断：

DeltaNorm = √(Σ error(e)) / TotalEnergy

其中 TotalEnergy 为系统总能量的平方根：

TotalEnergy = √(Σ |J(e)|²)

收敛条件为：

consecutive_converged_passes ≥ min_converged_passes

即连续若干轮的 DeltaNorm 都低于目标阈值（TargetMaxDeltaNorm）时，自适应迭代停止。每轮求解器还会检查当前误差与上一轮误差的变化量：

  • 如果变化量小于 TargetMaxDeltaNorm，该轮计为"收敛"
  • 如果连续收敛轮数达到设定值（通常为 2 轮），迭代停止
  • 如果达到 MaxAdaptiveRefinement 轮上限，强制停止

从实际仿真数据看，一次典型的 Q3D 仿真经历 5 轮自适应：

轮次三角面数目标细化数TotalEnergy
146504730.753
251166400.716
356287040.715
461927740.711
568148520.707

TotalEnergy 随轮次递减，说明解在逐渐收敛。

7. 总结

Q3D Extractor 的自适应网格加密基于一个核心思想：电流密度在单元边界上的不连续性度量误差。完整流程可总结为：

  1. 求解：PARDISO 解出边电流系数 I
  2. 重构：RT 基函数重构每个三角面的 J
  3. 检查：对每条边，比较自 J 和邻居 J 在边方向上的投影
  4. 量化：error = area × (Σ jump × len)²
  5. 选择：对数分桶 + PercentRefine 选取加密单元
  6. 收敛：DeltaNorm 低于阈值或达到上限轮数

误差公式统一为：

error = area × (Σ (Jself·dir + |dot|·Jneigh·dir) × edge_len)²
