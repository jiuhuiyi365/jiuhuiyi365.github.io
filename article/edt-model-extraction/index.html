1. 前置概念：Lac 与 Ldc 是什么

在电磁仿真中，电感是最常提取的参数之一。理解 Lac 和 Ldc 的区别是正确提取数据的前提。

参数全称含义典型应用场景
LdcDC Inductance直流偏置下的电感值，即静态电感功率电感设计、DC-DC 变换器
LacAC Inductance交流小信号下的增量电感 dΦ/di高频电路、信号完整性分析

两者的核心区别：

  • Ldc = Φ / I（磁链除以电流），反映的是"总"电感
  • Lac = dΦ / di（磁链对电流的微分），反映的是"增量"电感

当铁芯未饱和时，Lac ≈ Ldc；当铁芯进入饱和区后，Lac 会明显下降，而 Ldc 的下降趋势不同。因此在功率电感设计中，同时提取 Lac 和 Ldc 随电流变化的曲线非常重要。

TIP
在 Maxwell 中，Matrix 参数计算得到的是 Lac（增量电感），而通过 Flux Linkage / Current 可以手动计算 Ldc。

2. 仿真求解器的选择

根据你的模型类型和分析目标，选择合适的求解器：

求解器适用场景可提取的参数
Maxwell 2D/3D变压器、电感、电机等低频电磁器件Lac, Ldc, Flux Linkage, Force, Torque
HFSS天线、微波器件、高速互连等高频结构S 参数, Z 参数, Y 参数, 场分布
Q3D Extractor寄生参数提取（RLCG）R, L, C, G 矩阵

本教程以 Maxwell 为主，因为 Lac/Ldc 的提取主要在 Maxwell 中进行。

3. 从模型生成数据文件

3.1 Maxwell 导出 Solution Data

仿真完成后，按以下步骤导出数据：

方法一：通过 Results 面板

  1. 在左侧项目树中展开 Results
  2. 右键点击 Solution 或已有的报告
  3. 选择 Export → Solution Data
  4. 在弹出的对话框中：
    • Category：选择你要导出的量（如 Inductance、Flux Linkage）
    • Quantity：选择具体参数（如 L(Winding1,Winding1)）
    • Sweep：选择扫描变量（如 Current、Frequency）
    • Format：选择输出格式（CSV 或 Tab Delimited）
  5. 点击 Export 保存文件

方法二：通过 Create Report

  1. 右键 Results → Create Report
  2. 选择 Rectangular Plot（矩形图）
  3. 配置：
    • Category：Inductance
    • Quantity：L(Winding1,Winding1)
    • X Axis：选择扫描变量
  4. 点击 New Report 生成图表
  5. 在图表上右键 → Export → Export to CSV

3.2 HFSS 导出 S 参数 / Z 参数

对于 HFSS 仿真，导出流程类似：

  1. Results → Solution Data
  2. 选择 S Matrix 或 Z Matrix
  3. 选择端口组合（如 S(1,1)、S(2,1)）
  4. 选择格式：Real/Imaginary、Magnitude/Phase、dB
  5. 导出为 CSV
WARN
HFSS 中的 Z 参数可以用来计算等效电感：L = Im(Z) / (2πf)，但这只是等效值，不等同于 Maxwell 中的物理电感。

3.3 使用脚本批量导出

EDT 2019 R2 内置 IronPython 2.7 脚本引擎，可以自动化导出流程。在菜单栏 Tools → Script → Run Script 中执行。

TIP
不确定某个操作的脚本 API？EDT 2019 R2 自带宏录制功能：菜单栏 Tools → Script → Record Script，手动操作一遍，自动生成对应的 IronPython 脚本。这是学习 API 最快的方式。

IronPython

# Maxwell 批量导出 Solution Data
import os

oDesign = oProject.SetActiveDesign(“Maxwell3DDesign1”)
oModule = oDesign.GetModule(“Solutions”)


导出 Inductance 数据


export_path = r”E:\SimulationData\inductance.csv”
oModule.ExportSolutionData(
export_path,           # 输出路径
“Inductance”,          # Category
“L(Winding1,Winding1)”, # Quantity
[“Current:=”, [“All”]]  # 扫描范围
)


print(“Exported to: ” + export_path)

批量导出多个参数

# 批量导出多个参数到不同文件
oDesign = oProject.SetActiveDesign("Maxwell3DDesign1")
oModule = oDesign.GetModule("Solutions")

output_dir = r”E:\SimulationData”
params = [
(“Inductance”, “L(Winding1,Winding1)”, “Lac.csv”),
(“FluxLinkage”, “Flux(Winding1)”, “flux.csv”),
(“Current”, “I(Winding1)”, “current.csv”),
]


for category, quantity, filename in params:
filepath = os.path.join(output_dir, filename)
oModule.ExportSolutionData(
filepath,
category,
quantity,
[“Current:=”, [“All”]]
)
print(“Exported: ” + filename)

4. Lac 与 Ldc 曲线绘制

4.1 在 EDT 内部绘制

绘制 Lac 曲线（增量电感）：

  1. Results → Create Report
  2. Report Type 选择 Rectangular Plot
  3. 配置参数：
    • Domain：Sweep
    • Category：Inductance
    • Quantity：L(Winding1,Winding1)（这就是 Lac）
    • X Axis：Current
  4. 点击 New Report

绘制 Ldc 曲线（静态电感）：

Ldc 需要通过公式计算，因为 Maxwell 直接给出的是 Flux Linkage（磁链），需要手动除以电流：

  1. Results → Create Report
  2. 点击 Calculator（计算器按钮）
  3. 在表达式编辑器中输入：
    • 先选择 Flux Linkage → Flux(Winding1)
    • 再选择 Current → I(Winding1)
    • 组合为：Flux(Winding1) / I(Winding1)
  4. 或者直接在 Quantity 中输入表达式：
# Ldc 表达式
Flux(Winding1) / I(Winding1)

在同一张图上叠加 Lac 和 Ldc：

  1. 先创建 Lac 的报告
  2. 在已有的报告窗口中右键 → Add Trace
  3. 添加 Ldc 的表达式
  4. 两条曲线会显示在同一张图上，方便对比

4.2 用 Python 绘制外部曲线

如果需要更灵活的绘图，可以将数据导出后用 Python 处理：

Python

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

读取导出的 CSV 数据


假设 CSV 格式：Current, FluxLinkage, Inductance(Lac)


df = pd.read_csv(r”E:\SimulationData\inductance.csv”)


如果 CSV 中有 Flux Linkage 和 Current，可以计算 Ldc


if “Flux(Winding1)” in df.columns and “I(Winding1)” in df.columns:
df[“Ldc”] = df[“Flux(Winding1)”] / df[“I(Winding1)”]
df[“Lac”] = df[“L(Winding1,Winding1)“]


绘制曲线


fig, ax = plt.subplots(figsize=(10, 6))


ax.plot(df[“Current”], df[“Lac”] * 1e6, “b-o”, label=“Lac (增量电感)”, markersize=3)
ax.plot(df[“Current”], df[“Ldc”] * 1e6, “r-s”, label=“Ldc (静态电感)”, markersize=3)


ax.set_xlabel(“Current (A)”, fontsize=12)
ax.set_ylabel(“Inductance (μH)”, fontsize=12)
ax.set_title(“Lac vs Ldc 随电流变化曲线”, fontsize=14)
ax.legend(fontsize=11)
ax.grid(True, alpha=0.3)


plt.tight_layout()
plt.savefig(r”E:\SimulationData\Lac_Ldc_curve.png”, dpi=150)
plt.show()


print(“曲线已保存”)

TIP
注意单位转换：Maxwell 默认输出的电感单位是 Henries (H)，绘图时通常转换为 μH（乘以 1e6）或 mH（乘以 1e3）。

处理多组仿真数据（参数扫描）

import glob

# 读取多组仿真结果（如不同匝数的电感）
files = glob.glob(r"E:\SimulationData\N*.csv")

fig, ax = plt.subplots(figsize=(10, 6))

for f in sorted(files):
    df = pd.read_csv(f)
    label = f.split("\\")[-1].replace(".csv", "")  # 文件名作为标签
    ax.plot(df["Current"], df["Lac"] * 1e6, "-o", label=label, markersize=3)

ax.set_xlabel("Current (A)")
ax.set_ylabel("Lac (μH)")
ax.set_title("不同匝数下的 Lac 曲线对比")
ax.legend()

plt.tight_layout()
plt.show()

5. 模型几何导出

如果需要将 3D 几何模型导出到其他 CAD 软件：

  1. 在 3D Modeler 窗口中，选择 Modeler → Export
  2. 选择导出格式：
格式后缀用途
STEP.step / .stp通用 CAD 交换，SolidWorks / CATIA 等均可打开
ACIS.satAutoCAD 兼容
STL.stl3D 打印、网格处理
Parasolid.x_tSiemens NX、SolidWorks
IGES.iges老版 CAD 交换格式

导出整个模型还是选中的部分：

  • File → Export：导出整个模型
  • 先选中某些物体 → Modeler → Export Selection：只导出选中的部分
WARN
导出 STEP 时，Maxwell 的求解域（Region）和空气域通常不需要导出。在导出前可以临时隐藏或删除这些对象。

6. 详细日志与输出配置

仿真过程中，EDT 默认只显示基本的求解进度。如果你需要更详细的输出信息（如每一步的收敛数据、网格统计、矩阵参数等），可以通过以下方式配置。

6.1 Message Manager 窗口过滤

EDT 2019 R2 的 Message Manager 窗口（底部面板）自带过滤功能，可以按级别筛选消息：

  1. 确保 Message Manager 窗口可见：View → Message Manager（如果没有显示）
  2. 在 Message Manager 窗口顶部的工具栏中，有三个过滤按钮：
    • E（Error）— 显示错误信息
    • W（Warning）— 显示警告信息
    • I（Info）— 显示一般信息
  3. 默认情况下三个按钮都是激活的，点击可以切换开关
WARN
EDT 2019 R2 的 Tools → Options → General Options 中没有独立的 "Message Level" 下拉菜单。消息过滤主要通过 Message Manager 窗口的工具栏按钮来控制。

通过注册表开启调试消息输出（修改后需重启 EDT）：

Windows Registry Editor Version 5.00

; 开启 Twin Builder / Simplorer 的调试和跟踪消息
[HKEY_CURRENT_USER\Software\Ansoft\Twin Builder]
"DebugMessages"=dword:00000001
"TraceMessages"=dword:00000001
"WarningMessages"=dword:00000001

; 0 = 关闭, 1 = 开启
TIP
对于 HFSS 和 Maxwell，更实用的做法是通过 Analysis Setup 增加输出详细程度（见 6.2 节），以及在仿真后查看 Solution Data 中的收敛和 Profile 信息（见 6.3 节）。

6.2 Analysis Setup 详细输出

在 Analysis Setup 中可以控制仿真过程的输出细节：

Maxwell 自适应求解设置：

  1. 双击项目树中的 Analysis → Setup1
  2. 在 Convergence 选项卡中：
    • Maximum Number of Passes：最大迭代次数（默认 10，调试时可设到 20+）
    • Maximum Delta Energy (%)：能量收敛精度（默认 1%，可收紧到 0.1%）
    • Percent Refinement：每次迭代的网格细化比例（默认 30%）
  3. 在 Solve 选项卡中：
    • 勾选 Save Fields：保存每一步的场分布（便于查看收敛过程）
    • 勾选 Copy geometrically equivalent meshes：对参数扫描复用网格

HFSS 自适应求解设置：

  1. 双击 Analysis → Setup1
  2. General 选项卡：
    • Maximum Number of Passes：最大迭代次数
    • Maximum Delta S：S 参数收敛标准（默认 0.02，可收紧到 0.005）
  3. Options 选项卡：
    • 勾选 Save Fields
    • 勾选 Save Rad Fields Only（天线场景）

6.3 Simulation Profile 与收敛信息

仿真完成后，查看详细的 Profile 信息：

  1. 右键 Results → Solution Data
  2. 切换到 Profile 选项卡 — 显示每一步的：
    • 网格单元数（Number of Elements）
    • 求解时间
    • 内存占用
  3. 切换到 Convergence 选项卡 — 显示：
    • 每一步的收敛误差曲线
    • 能量 / S 参数的变化趋势

通过脚本获取收敛数据：

IronPython

# 获取仿真 Profile 信息
oDesign = oProject.SetActiveDesign("Maxwell3DDesign1")
oModule = oDesign.GetModule("Solutions")

获取自适应收敛信息


convergence = oModule.GetConvergenceData(“Setup1 : LastAdaptive”)
print(”=== Convergence Data ===”)
print(convergence)


获取 Profile（求解统计）


profile = oModule.GetProfileData(“Setup1 : LastAdaptive”)
print(”=== Profile Data ===”)
print(profile)

使用脚本时，可以通过多种方式输出调试信息到不同位置：

IronPython — 输出到 Message Manager

# oDesktop 是 EDT 的全局对象，始终可用

普通信息（蓝色）


oDesktop.AddMessage("", "", 0, “This is an info message”)


警告（黄色）


oDesktop.AddMessage("", "", 1, “This is a warning message”)


错误（红色）


oDesktop.AddMessage("", "", 2, “This is an error message”)


级别说明：


0 = Informational


1 = Warning


2 = Error


3 = Fatal

IronPython — 输出详细仿真数据

# 遍历并打印所有求解结果
oDesign = oProject.SetActiveDesign("Maxwell3DDesign1")
oModule = oDesign.GetModule("Solutions")

获取所有可用的求解数据


solutions = oModule.GetSolutions(“Setup1 : LastAdaptive”)


打印到消息管理器


for sol in solutions:
oDesktop.AddMessage("", "", 0, “Solution: ” + str(sol))


导出收敛历史到文本文件


import os
log_path = os.path.join(os.environ[“TEMP”], “maxwell_convergence.txt”)
oModule.ExportConvergenceData(“Setup1”, log_path)
oDesktop.AddMessage("", "", 0, “Convergence log saved to: ” + log_path)

输出到外部日志文件

# 将仿真过程中的关键数据写入外部日志
import os
import datetime

log_file = r”E:\SimulationData\sim_log.txt”


def log(msg):
timestamp = datetime.datetime.now().strftime(“%Y-%m-%d %H:%M:%S”)
line = ”[%s] %s” % (timestamp, msg)


# 写入文件
with open(log_file, "a") as f:
    f.write(line + "\n")

# 同时输出到 Message Manager
oDesktop.AddMessage("", "", 0, line)


使用示例


log(“Simulation started”)
log(“Design: ” + oDesign.GetName())
log(“Setup: Setup1”)
log(“Waiting for solution…”)


… 仿真完成后 …


log(“Solution completed”)

6.5 日志文件位置

EDT 在运行过程中会自动生成多个日志文件，默认位置：

文件类型路径内容
项目日志<project_dir>\<project>.aedt.lock项目锁定信息
求解器日志<project_dir>\<design>.solv求解器配置和状态
网格日志<project_dir>\<design>.mesh网格划分详情
收敛日志<project_dir>\<design>.conv自适应收敛历史
Profile 日志<project_dir>\<design>.prof求解时间和内存统计
错误日志<project_dir>\<design>.err错误和警告信息

快速打开项目目录：

  1. 在项目树顶部右键项目名称
  2. 选择 Open Directory（或 Project Path）
  3. 即可看到所有日志文件
WARN
.solv、.mesh、.conv 等文件是二进制格式，不能直接用文本编辑器打开。需要通过 EDT 的 Results → Solution Data 面板查看，或使用脚本导出为可读格式。
IMPORTANT
如果仿真中途崩溃或异常退出，务必检查 .err 文件和项目目录下的 scheduler.log，它们通常包含崩溃前的最后状态信息，是排查问题的关键线索。

7. 常见问题与注意事项

Q1：导出的 CSV 打开是乱码？

EDT 导出的 CSV 可能使用 UTF-16 编码。用 Excel 打开时选择"数据 → 自文本"，手动指定编码为 UTF-16。

Q2：Lac 出现负值或异常跳变？

通常是因为铁芯进入了深度饱和区。检查：

  • 材料的 B-H 曲线是否覆盖了实际工作点
  • 电流扫描步长是否太大，尝试减小步长

Q3：如何提取多绕组变压器的互感？

在 Maxwell 中使用 Matrix 参数：

  1. Maxwell → Parameters → Assign → Matrix
  2. 勾选所有绕组
  3. 仿真完成后，在 Results 中查看 M(Winding1,Winding2)（互感）

Q4：脚本导出报错 oProject is not defined？

脚本需要在 EDT 的脚本环境中运行，不能在外部 Python IDE 中直接执行。使用 Tools → Script → Run Script 加载 .py 文件。

Q5：脚本报错 SyntaxError: invalid syntax（f-string 相关）？

EDT 2019 R2 的脚本引擎是 IronPython 2.7，不支持 Python 3 的 f-string 语法。需要将：

# 错误写法（Python 3）
msg = f"Design: {name}, Setup: {setup}"

正确写法（IronPython 2.7 兼容）


msg = “Design: %s, Setup: %s” % (name, setup)


或


msg = “Design: {}, Setup: {}“.format(name, setup)

IMPORTANT
仿真结果的准确性取决于网格质量。在关键区域（如气隙、绕组附近）使用 Mesh Operation → Assign 手动加密网格，可以显著提高 Lac / Ldc 的计算精度。
